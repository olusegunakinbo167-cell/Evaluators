# Evaluators
A TypeScript/Node.js tool that implements a full rubric-based AI code ranking system.
# 1. Unzip the project
unzip ai-code-evaluator.zip
cd ai-code-evaluator

# 2. Initialize git
git init
git add .
git commit -m "feat: initial release — AI code quality evaluator for RLHF workflows"

# 3. Create a new repo on GitHub named: ai-code-evaluator
# Then push:
git remote add origin https://github.com/YOUR_USERNAME/ai-code-evaluator.git
git branch -M main
git push -u origin main
